## 📖 常用功能速查

### 🔍 搜索异议
- 在搜索框输入关键词
- 支持：异议标题、关键词、分类、描述

### 📋 查看完整方案
- 点击左侧异议列表
- 右侧显示：分析、方案、数据、案例、技巧

### 📋 复制话术
- 点击"📋 复制方案"按钮
- 自动复制到剪贴板
- 粘贴到销售工具中

### 🏷️ 分类筛选
- 点击上方分类标签
- 快速查看特定类型的异议

### 💾 导出数据
```python
from objection_assistant import ObjectionAssistant
assistant = ObjectionAssistant()

# 导出 Excel
assistant.export_to_excel("异议库.xlsx")

# 导出 JSON
assistant.export_to_json("异议库.json")
```

### 🤖 AI 智能匹配
```python
# 根据客户表述推荐异议
result = assistant.smart_match("客户说价格太贵了")

# 系统推荐 Top 3 相关异议
for rec in result['recommendations']:
    print(rec['objection']['title'])
```

## 🎯 典型工作流程

### 销售处理客户异议的流程

```
1. 客户提出异议
   ↓
2. 销售在异议助手中搜索
   关键词示例：价格、效果、竞品、时间
   ↓
3. 查看异议详情和方案
   包含：3-5 个回复方案、数据支撑、成功案例
   ↓
4. 复制最合适的话术
   一键复制到剪贴板
   ↓
5. 在销售工具中粘贴话术
   微信、钉钉、飞书等
   ↓
6. 与客户沟通
   使用专业的、经过验证的话术
   ↓
7. 可选：评分反馈
   告诉系统这个方案是否有效
```
