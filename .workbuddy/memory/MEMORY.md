## 在线网站部署

**版本 2.1** - 新增公网访问能力

### 部署方案（三选一）

1. **本地启动**（开发）
   - 命令：`bash run_web.sh`
   - 访问：http://127.0.0.1:5000
   - 用途：快速测试、内部共享

2. **GitHub Pages**（推荐）
   - 方式：上传到 GitHub + 启用 Pages
   - 成本：完全免费
   - 访问：https://{用户名}.github.io/objection-assistant
   - 用途：公网共享、永久免费

3. **云服务器**（企业级）
   - 推荐：Render、Railway、阿里云
   - 成本：¥0-99/月
   - 用途：高可用、大流量

### 新增文件

- `web_app.py` - Flask Web 应用（支持 API）
- `index.html` - 静态版本（可部署到 GitHub Pages）
- `run_web.sh` - 启动脚本
- `部署指南.md` - 完整部署说明
- `GitHub_Pages_部署.md` - GitHub Pages 教程
- `在线网站快速开始.md` - 快速上手指南

### API 端点

- `GET /` - 主页
- `GET /api/objections` - 获取所有异议
- `GET /api/objections/<id>` - 获取单个异议
- `GET /api/search?q=...` - 搜索异议
- `GET /api/stats` - 获取统计信息
