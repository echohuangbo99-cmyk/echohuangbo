# 🎯 松鼠AI 销售异议查询助手 - 使用指南

## 📋 概述

这是一个为松鼠AI门店销售人员设计的 **AI 驱动异议查询系统**，帮助销售快速查找和应对客户异议。

### 核心功能
✅ **智能搜索** - 快速查找相关异议  
✅ **分类浏览** - 按异议类型分类查看  
✅ **完整方案** - 每个异议配备5-7个回复方案  
✅ **数据支撑** - 包含客户数据、成功案例  
✅ **一键复制** - 快速复制话术到销售话术库  
✅ **离线可用** - 网页版可离线使用  
✅ **智能匹配** - AI 根据客户表述自动推荐异议

## 🚀 快速开始

### 方式一：使用网页版（推荐）

1. **打开应用**
   - 在电脑浏览器中打开 `objection_assistant.html`
   - 或部署到服务器后通过 URL 访问

2. **查找异议**
   ```
   方法1：关键词搜索
   - 在搜索框输入关键词（如"价格"、"效果"、"竞品"）
   - 系统自动匹配相关异议
   
   方法2：分类浏览
   - 点击右上方的分类标签快速筛选
   - 价格相关、产品效果、竞争对比等
   
   方法3：浏览全部
   - 点击"全部"标签查看所有异议
   ```

3. **查看详情**
   - 点击左侧异议列表中的任意项目
   - 右侧会显示完整的解决方案

4. **复制话术**
   - 点击"📋 复制方案"按钮
   - 方案会自动复制到剪贴板
   - 可直接粘贴到微信、钉钉等工具

### 方式二：使用 Python 后端（数据管理）

```bash
# 1. 安装依赖（可选）
pip install openpyxl  # 如果需要 Excel 导出功能

# 2. 在 Python 环境中运行
python objection_assistant.py

# 3. 在代码中调用
from objection_assistant import ObjectionAssistant

assistant = ObjectionAssistant()

# 搜索异议
results = assistant.search_objections("价格")

# 智能匹配
matched = assistant.smart_match("客户说太贵了")

# 导出数据
assistant.export_to_excel("异议库.xlsx")
```

## 🔍 高级功能

### 1. 智能匹配（AI 推荐）

系统会自动根据客户表述推荐最相关的异议：

```python
# 使用示例
assistant = ObjectionAssistant()

# 输入客户原话
customer_objection = "我看你们的产品和竞品差不多，为什么我要选你们？"

# AI 智能匹配
result = assistant.smart_match(customer_objection)

# 系统自动推荐Top 3相关异议
for recommendation in result['recommendations']:
    print(recommendation['objection']['title'])  # 推荐的异议标题
    print(recommendation['matchReason'])  # 匹配理由
```

### 2. 数据统计

查看异议库的使用统计数据：

```python
stats = assistant.get_statistics()

# 输出示例：
# {
#   'totalObjections': 28,
#   'totalCategories': 8,
#   'totalUsage': 1523,  # 总查询次数
#   'averageRating': 4.6,  # 平均评分
#   'categoryDistribution': {'价格相关': 3, '产品效果': 2, ...},
#   'mostUsed': [...],  # 最常查询的异议
#   'topRated': [...]  # 用户评分最高的异议
# }
```

### 3. 数据导出

将异议库导出为 Excel 或 JSON 格式：

```python
# 导出为 Excel（便于分享和打印）
assistant.export_to_excel("松鼠AI异议库.xlsx")

# 导出为 JSON（便于系统集成）
assistant.export_to_json("异议库.json")
```

### 4. 编辑和扩展异议库

```python
# 添加新异议
new_objection = {
    'title': '新的客户异议',
    'category': '价格相关',
    'keywords': ['关键词1', '关键词2'],
    'description': '异议描述',
    'analysis': '为什么客户会有这个异议',
    'solutions': ['方案1', '方案2', '方案3'],
    'dataPoints': ['数据点1', '数据点2'],
    'counterExamples': ['案例1', '案例2'],
    'tips': ['技巧1', '技巧2']
}

result = assistant.add_objection(new_objection)
print(f"新异议已添加，ID: {result['id']}")

# 更新异议
assistant.update_objection(obj_id=1, updates={
    'solutions': ['更新的方案1', '更新的方案2']
})

# 删除异议
assistant.delete_objection(obj_id=1)
```

## 🎓 销售技巧总结

### 通用原则

1. **不否定，要同意**
   > "我理解您的顾虑，这确实是很多家长最关心的问题..."

2. **用数据说话**
   > 具体数据 > 笼统承诺

3. **讲对比，不贬低**
   > 强调自己的优势，而不是批评竞品

4. **提供选择**
   > 提供多个方案，让客户选择

5. **制造紧迫感**
   > "目前我们正在进行优惠活动..."

### 按异议类型的策略

| 异议类型 | 关键策略 | 主要话术 |
|---------|--------|--------|
| **价格** | 强调ROI | "按年均算下来，我们成本最低，效果最优" |
| **效果** | 讲数据 | "平均提分25分，远超行业水平" |
| **对比** | 差异化 | "我们的知识图谱最完整，服务最专业" |
| **时间** | 碎片化 | "每天只需15-20分钟，利用碎片时间" |
| **信任** | 展示认证 | "通过ISO认证，数据安全有保障" |

## ❓ 常见问题

**Q1: 怎样快速找到某个异议？**
> A: 使用搜索框输入关键词，系统会实时搜索匹配。

**Q2: 能否自定义添加新异议？**
> A: 可以。使用 Python 脚本添加新异议，或直接编辑 HTML 中的 objectionsData。

**Q3: 如何团队协作维护异议库？**
> A: 建议部署到云端，所有销售共用同一个库，后台管理员定期更新。

**Q4: 可以离线使用吗？**
> A: 可以。网页版是纯前端应用，下载 HTML 文件即可离线使用。

**Q5: 如何导出异议库分享给团队？**
> A: 使用 export_to_excel() 函数，导出为 Excel 文件便于分享。
